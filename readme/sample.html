KBC Bootstrap CSSサンプル

このページではKBC実行委員会のWebページを構成するための補助ライブラリ「KBC Bootstrap」を使ったCSSでのWebページの構成方法を説明しています。本ページでは全ての紹介要素について実例を掲載していますので、もし説明で具体的な書き方を理解できなければ、このページのソースコードを見て使い方を学んで下さい。
また、このようなレイアウト用のクラス群はデザインを統一するため、なるべく各ページで用いるようにするべきです。ここで用意されているものと同じ目的の要素を独自に定義するとWebページ全体の見た目が悪くなるだけでなく、将来的なコードの保守性や新しいページを作成する際のコードの可搬性を著しく低下させてしまいます。

※ 本ページで示すサンプルではそれがサンプルであると分かりやすいように、この文章が書かれているPタグと同じように上下左右に多く余白を取っています。そのため、実際に使用する場合とは表示される大きさが異なりますので、注意して下さい。

ヘッダー

KBC Bootstrapには豊富なカラーバリエーションのヘッダーが標準で用意されています。
これらを利用するにはH2タグまたはH3タグのクラスに「caption-[色名]」を追加します。以下はH2タグに「caption-red」クラスを追加した場合の例です。

H2.caption-red

また、以下はH3タグでの全てのカラーバリエーションの例です。

H3.caption-black

H3.caption-red

H3.caption-green

H3.caption-blue

H3.caption-yellow

H3.caption-white

H3.caption-orange

H3.caption-purple

H3.caption-brown

さらに、H4タグはH2タグやH3タグで区切られたコンテンツ内の小見出しとして使用することができます。

H4

なお、これらのヘッダータグで区切ったコンテンツ内の文章はインデントを合わせる必要があるので、必ずPタグで囲みます。まさに今読まれているこの文章もPタグで囲まれています。

ボタン

KBC BootstrapではTwitter Bootstrapで用意されているボタンと同じ記法で、KBCのWebページ用に余白などの調節されたものを使用することができます。Twitter Bootstrapの記法では、ボタンにしたい要素(主にAタグ)のクラスに「btn」と「btn-[種類名]」を追加します。以下は作成できる全ての種類のボタンの例です。

A.btn.btn-default A.btn.btn-primary A.btn.btn-success A.btn.btn-info A.btn.btn-warning A.btn.btn-danger

サムネイル

写真をメインに載せるのではなく、イメージとして文章の横に画像を表示したい場合にはKBC Bootstrapのサムネイルレイアウトが便利です。サムネイルレイアウトではmiddle(画像250px)、small(画像200px)、mini(画像150px)の3種類の大きさに加え、画像を左右どちらに表示するかを選択することができます。サムネイルレイアウトはDIVタグで囲ったPタグと、そのDIVタグの中でさらにSPANタグに囲まれたIMGタグによって構成されます。
