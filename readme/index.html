READMEページへようこそ!

ここはKBC実行委員会のメンバーが本Webページを編集するための情報をまとめたページです。Web担当を行うメンバーは必ず本ページで示されている内容を読み、よく理解した上でWebの編集を行って下さい。
また、このページはKBC実行委員会のメンバーだけに見えるように管理されるべきです。アップロードされている場合には必ず認証を掛けるようにして下さい。

開発環境の構築

KBC実行委員会のWebページではページの可搬性を高めるために、全てのパスを絶対パスで記述します。そのため、Webページの実際の見た目を確認しながらローカル環境のみで開発を進めるためにはいくつかの準備が必要となります。そうした準備の仕方については開発環境の構築方法を参照して下さい。

KBC Bootstrapの使用

KBC実行委員会のWebページはjQueryおよびTwitter BootstrapをベースとしたJavaScript/CSSライブラリである「KBC Bootstrap」によって構成されています。

KBC Bootstrap CSS

適切なクラスの指定によって、コンテンツを統一されたデザインで簡単に仕上げることができます。指定できるクラスの一覧とプレビュー、使い方の詳細についてはKBC Bootstrap CSSサンプルを参照して下さい。

KBC Bootstrap JS

KBC Bootstrapにより、Webページの簡単な更新のほとんどはHTMLファイルを記述せず、JSONファイルを編集するだけで行うことができます。具体的な編集の仕方についてはKBC Bootstrap JS JSON記法を参照して下さい。

ディレクトリ構造の管理

KBC実行委員会のWebページは将来的にファイルが増大したとしても、その検索・維持・管理に大きなコストの掛かることがないよう、単純かつ明確なディレクトリ構造を持っています。そのため、新たなHTMLファイルや画像ファイル、CSSファイル、JavaScriptファイルに加え、応募書類などのダウンロードファイルを追加する場合は必ずディレクトリ構造解説ページを参照し、その構造に従った名前付け、ディレクトリ作成を行って下さい。

より高度な開発

Webページの更新など、比較的簡単な編集であればKBC Bootstrap JSに基づいてJSONファイルを編集するだけで事足りるでしょう。しかし、今後大きなプロジェクトの改変などがあった場合にはWebページのスタイルや形式を変更するために、既存のHTMLファイルやKBC Bootstrap自体を書き換える必要が出てくるかもしれません。KBC Bootstrap Developmentページはそうした要求に答えるためのKBC Bootstrapの解説ページです。
